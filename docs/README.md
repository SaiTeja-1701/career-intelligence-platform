# AI-Powered Career Intelligence & Skill Gap Platform

## Overview
This project is an end-to-end AI-driven career guidance platform that evaluates resumes, predicts job fit, identifies skill gaps, and provides explainable career recommendations using machine learning and large language models.

## Key Features
- Resume parsing and skill extraction
- Job-fit prediction
- Skill gap analysis
- Skill demand forecasting
- Explainable AI chatbot
- Role-based dashboards

## Tech Stack
- Frontend: React (MERN)
- Backend: Node.js, Express
- ML Service: FastAPI, Python
- Database: MongoDB Atlas
- AI/ML: NLP, Time-Series Forecasting, RAG
